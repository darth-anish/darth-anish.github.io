Project I
Face Recognition

Face Recognition is a classification task of recognizing the class of a given face among the three trained classes. The deep learning model for the task comprises of CNN, Maxpooling and Dense layers. The model is saved in h5py format and can be used to predict any given face image among the three faces.

Project II
Text Classification

Text Classification is a sentiment classification task. The model built comprises of Embedding, Bidirectional LSTM, Dropout and Dense layers. It can be used to predict any given text as postive or negative sentiment

Project III
Person Detection

Person Detection is a deep learning prototype for detection of a person based on YOLO algorithm for object dection. The model implements the trained model darketnet using OpenCV which takes real time video as feed to the network which detects the person in it. The another task done was the count of persons detected in an interval and store it in a MongoDB database.
