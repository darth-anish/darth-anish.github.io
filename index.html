Project I
Predicting Risk of 30-Day Readmission for Diabetic Patients

Prediction Risk of 30-Day Readmission is a capstone project which predicts the likelihood of a patient being readmitted within 30 days. Various machine learning algorithms are modelled with Hypertuned random forest being the winner model. The permutation feature importance and clustering analysis are combined to provide a 5 type patient profile which could classify any patient into one of these profiles, providing hospitals and doctors preliminary idea about specific treatment required. Project IV
News Classification using RNN(NLP)

News Classification project is a NLP project which classifies a news into one of the five news categories. The classification task is done by a LSTM based RNN with 50 units on it. The first input layer is embedding layer while the output layer is dense layer with 5 units since 5 classes are there to classify.

Project V
Person Detection

Person Detection is a deep learning prototype for detection of a person based on YOLO algorithm for object dection. The model implements the trained model darketnet using OpenCV which takes real time video as feed to the network which detects the person in it. The another task done was the count of persons detected in an interval and store it in a MongoDB database.
